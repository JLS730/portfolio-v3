Some of My Work
Git-A-Pup

Git-A-Pup is a dog adoption website that I have created that is powered by PetFinder API. Within this website, you can search for dogs and shelters in your area to discover dogs you are interested in. This website also uses Dog API, just in case you need more information about specific dog breeds.

gitMusic

gitMusic is an online music store where you can purchase some of your favorite albums! From undebatable classics to exemplary modern hits, this store has everything to offer! This store uses Commerce.js to handle the shopping cart and it's also made using React, Sass and React Router.

FireMedia

FireMedia is a movie database website where you're able to view information about upcoming movies, watch trailers and behind the scenes videos, and create a watch later list to keep track of movies you would like to view later. FireMedia uses firebase's authentication with it's realtime database, alongside The Movie Database API.

Fire! (Game Remake)

This is a remake of a game named Fire from Mr. Game and Watch Gallery 4. The objective is to save as many people as you can from a burning building by juggling them on a stretcher until they reach the ambulance. This game was made with custom sprites, CSS animations and vanilla JavaScript, using an update loop.
